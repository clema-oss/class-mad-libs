# class-mad-libs

There will be a lot of contributors... I hope this goes well.

## How to Contribute

1. **Clone this repository** to your computer

3. **Edit your Mad Lib file** - Choose the file named after you and add your Mad Lib from JuiceMind

4. **Commit your changes** with a clear message

5. **Push to main**

6. **Create a Pull Request** to suggest adding your changes to the main project


## Reminders: 
- Write clear concise commit messages


## Goals:
- Practice the clone-branch-edit-add-commit-push workflow

- Create and review Pull Requests
